Kimai - jQuery Mobile Interface

This repository contains a mobile optimized interface of Kimai. It's a small website that lets you start and stop your time record from mobile devices.

It's built with jQuery Mobile and works as a developer demonstration for the Kimai Remote API. For now it has the following features:

  • log in
  • read projects
  • read customers
  • read tasks
  • start the timer
  • stop the timer

It's a very lightweight solution made with only one HTML script, a couple of Javascript files and some spicy CSS sugar on top.

Please note, that this is the first (experimental) release and many features of the complete version are missing. We don't have plans to add more features soon, but we'd like to hear your thoughts and feature requests at the issue tracker.

Installation

To install this mobile interface:

  • copy all the files into a sub-directory of your Kimai installation (e.g. mobile/)
  • point your browser to http://yourKimaiInstallation/mobile/
  • log in with your existing Kimai account

Compatibility

This mobile time tracking solution only works with Kimai revision 1353 and later.
